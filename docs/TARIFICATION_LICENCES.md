## 5. STRATÉGIE DE LANCEMENT

### Phase 1 — Validation (Mois 1-3)

- 3-5 formations pilotes en essai gratuit (30 jours)
- Collecte de feedback et ajustements
- Conversion en licences annuelles CSPS/CMA

### Phase 2 — Commercialisation (Mois 4-12)

- Partenariat avec la Direction Générale des Formations Sanitaires (DGFS)
- Démonstrations dans les CHR régionaux
- Kit commercial : brochure, vidéo démo, témoignages pilotes
- Tarif promotionnel de lancement : -20% la première année

### Phase 3 — Expansion (Année 2+)

- Extension aux pays voisins (Mali, Niger, Togo, Bénin)
- Ajout de modules : gestion des stocks, facturation, RH
- API ouverte pour intégrateurs tiers
- Application mobile native (optionnel)
